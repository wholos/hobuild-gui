# hobuild-gui
[![C++](https://img.shields.io/badge/C++-%2300599C.svg?logo=c%2B%2B&logoColor=white)](#)

Hobuild - GUI!

Program for creating a sample ```build.hob``` in the selected directory.

**How to build?**
``` sh
chmod +x build.sh
./build.sh
```

**How to install?**
AUR (ArchLinux):
``` sh
yay -S hobuild-gui
```

Any unix systems:
```
cp hobuild-gui /usr/bin/hobuild-gui
```
